Website Installations

This is my website development and deployment portfolio page. I've installed several sites on my server so you can get a better idea of some of my expertise in this area. If you do happen to be running Wordpress, using either a built-in theme or one of the myriad of custom themes available in the marketplace today, I can help you get it installed, configured and updated to suit your needs and requirements.

If you have a completely custom website, written in PHP or Python, I can assist you with making changes to that as well; or, we can simply take your existing out-of-date site and get it modernized so that it works with the latest devices and browsers.
